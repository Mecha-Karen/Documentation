Roll
====
Karen rolls a dice between herself or with another member selected with you. Whoever gets the highest wins!
 
Parameters
----------
**Member [Optional]**

The member who you like to roll the dice against, by default its Karen herself!
 
**Layout:**
::
    -Roll [member]
 
Permissions
-----------
**Bot**
::
    Embed Links
 
**Author**
::
    Send Messages
 
Cooldown
--------
10 seconds.
