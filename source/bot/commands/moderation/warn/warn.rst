Warn
====

Add a warn to a specified user

Parameters
----------
**Member [Required]**

The member to add the warn to

**Reason [Optional]**

The reason for the warn - Defaults to "Wasn't Provided"

**Layout:**
::
	-Warn <Member> [Reason="Wasn't Provided"]

Permissions
-----------
**Bot**
::
	Embed Links
	Send Messages

**Author**
::
	Manage Messages

Cooldown
--------
5 Seconds - Per User
